===Install: node.js
	(Only for creating new app)Express http://expressjs.com/en/starter/installing.html

===Starting the app

1. Open CMD and go to the project's workplace directory and 
	type "node install" to install necessary modules from package.json

2. In the same CMD and directory, type "node webapp"

3. Open web browser and type the url "localhost:3000"

4. Done
	
	
(optional) If src files have been modified, it has to be gulped.
	cmd: "gulp bundle"
Gulp must be installed globally, "npm install --g gulp" and there should be a gulpfile.js
that has the code to compile.
